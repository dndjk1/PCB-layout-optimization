# Week 1 Report Outline

## 1. Problem Definition

- Input: `.nodes`, `.nets`, and `.pl` dataset files.
- Output: optimized `.pl` files in the same format as the input layout.
- Objective: minimize total HPWL.
- Constraints: boundary legality and minimum 2-unit spacing between components.

## 2. Data Format

### `.nodes`

Records component name, width, and height.

### `.nets`

Records net membership and pin offsets relative to component centers.

### `.pl`

Records component lower-left coordinates and orientation.

## 3. Evaluation Metric

For each net:

```text
HPWL = (max_x - min_x) + (max_y - min_y)
```

Total HPWL is the sum of HPWL over all nets.
