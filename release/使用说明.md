# PCBLayoutTool 使用说明

双击 `PCBLayoutTool.exe` 后，程序会启动本地 Web UI，并自动打开浏览器。

如果浏览器没有自动打开，可以在控制台窗口中查看类似下面的地址：

```text
PCB layout UI running at http://127.0.0.1:8777/
```

复制该地址到浏览器即可。

使用时请保持控制台窗口不要关闭；关闭窗口后，本地 UI 服务也会停止。

主要快捷键：

- 左键空白拖动：框选多个器件。
- 左键拖动选中器件：移动器件。
- 中键或右键拖动：移动视角。
- Ctrl+R：右转选中器件。
- Ctrl+空格：锁定或解锁选中器件。
- Ctrl+Z：撤回上一步布局操作。
